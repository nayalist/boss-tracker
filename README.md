Lightweight addon for WoW WotLK 3.3.5a AzerothCore client that tracks defeated bosses
in current instance (dungeon/raid) and timestamps for each kill.

<img width="302" height="348" alt="image" src="https://github.com/user-attachments/assets/1504663b-b429-474a-b7d4-b570a47eaf08" />


-Tracking does not persist if player leaves an instance before all bosses are cleared.
-Boss lists for Scarlet Monastery and Dire Maul East preload with full merged boss
lists until player kills an enemy in an uniquely identifiable subzone then boss lists
reduce to correct display.
-Violet hold shows full boss list even though only 3 will be spawned.
